## Getting Started in 3 Steps

### 1. Download
Select the **Standalone ZIP** from the link above. The package is approximately 170MB and self-contained.

### 2. Extract
Extract the downloaded ZIP to your preferred application directory (e.g., C:\EduCore).

### 3. Launch
Open the folder and launch **EduCore.exe**. The system will guide you through the initial configuration of your school's profile.
